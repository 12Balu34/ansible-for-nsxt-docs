nsxt_upgrade_prechecks -- Execute pre-upgrade checks
++++++++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 2.7

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Run pre-defined checks to identify potential issues which can be encountered during an upgrade or can cause an upgrade to fail.
- The results of the checks are added to the respective upgrade units aggregate-info.
- The progress and status of operation is part of upgrade status summary of individual components.

Examples
--------

.. code-block:: yaml+jinja

      nsxt_upgrade_prechecks:
          hostname: "10.192.167.137"
          username: "admin"
          password: "Admin!23Admin"
          validate_certs: False
          state: 'present'
