nsxt_virtual_ip -- Sets and clears cluster virtual IP address
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 2.7

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Sets the cluster virtual IP address.
- Note, all nodes in the management cluster must be in the same subnet.
- If not, a 409 CONFLICT status is returned.

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Adds cluster virtual IP address
      nsxt_virtual_ip:
          hostname: "10.192.167.137"
          username: "admin"
          password: "Admin!23Admin"
          validate_certs: False
          virtual_ip_address: "10.192.167.141"
          state: present
