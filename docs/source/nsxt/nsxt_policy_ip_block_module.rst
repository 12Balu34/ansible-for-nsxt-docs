nsxt_policy_ip_block -- Create or Delete a Policy IP Block
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 2.8

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Creates or deletes a Policy IP Block.
- Required attributes include id and display_name.

Examples
--------

.. code-block:: yaml+jinja

    
    - name: create IP Block
      nsxt_policy_ip_block:
        hostname: "10.10.10.10"
        nsx_cert_path: /root/com.vmware.nsx.ncp/nsx.crt
        nsx_key_path: /root/com.vmware.nsx.ncp/nsx.key
        validate_certs: False
        id: test-ip-blk
        display_name: test-ip-blk
        state: "present"
        cidr: "192.168.0.0/16"
