nsxt_transport_zones -- Create a Transport Zone
+++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 2.7

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Creates a new transport zone.
- The required parameters are host_switch_name and transport_type (OVERLAY or VLAN).
- The optional parameters are description and display_name.

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Create transport zone
      nsxt_transport_zones:
        hostname: "10.192.167.137"
        username: "admin"
        password: "Admin!23Admin"
        validate_certs: False
        resource_type: "TransportZone"
        display_name: "TZ1"
        description: "NSX configured Test Transport Zone"
        transport_type: "OVERLAY"
        host_switch_name: "hostswitch4"
        state: "present"
