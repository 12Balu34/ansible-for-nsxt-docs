nsxt_ip_blocks -- Create a new IP address block
+++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 2.7

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Creates a new IPv4 address block using the specified cidr.
- cidr is a required parameter.
- display_name & description are optional parameters.

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Create a new IP address block
      nsxt_ip_blocks:
        hostname: "10.192.167.137"
        username: "admin"
        password: "Admin!23Admin"
        validate_certs: False
        display_name: "IPBlock-Tenant-1"
        description: "IPBlock-Tenant-1 Description"
        cidr: "192.168.0.0/16"
        state: present
