nsxt_logical_routers -- Create a Logical Router
+++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 2.7

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Creates a logical router. The required parameters are router_type (TIER0 or TIER1) and edge_cluster_id (TIER0 only).
- Optional parameters include internal and external transit network addresses.

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Create a Logical Router
      nsxt_logical_routers:
          hostname: "10.192.167.137"
          username: "admin"
          password: "Admin!23Admin"
          validate_certs: False
          resource_type: LogicalRouter
          description: "Router West"
          display_name: "tier-0"
          edge_cluster_name: edge-cluster-1
          router_type: TIER0
          high_availability_mode: ACTIVE_ACTIVE
          state: present
